\section{AI-tool disclosure}
\label{app:ai}

The author used \emph{Claude Code} (Anthropic; models: Opus 4.7 and
Sonnet 4.6) as a coding and writing assistant throughout the project.
AI was used for: (i) code scaffolding (training-loop boilerplate, SLURM
scripts, Gradio app, the rolling-backtest cron, the validation script,
the bootstrap utility, the multi-window experiment runner, the figure
renderers); (ii) figure-rendering code (matplotlib generators in
\texttt{scripts/figures/}); (iii) \LaTeX{} boilerplate
(\texttt{preamble.tex}, citations); (iv) prose copy-editing on this
paper.

AI was \emph{not} used for: research design, model architecture
choices, training-result interpretation, the four geographic-orientation
correctness checks (Appendix~\ref{app:attention}), the BTM-solar
correlation hypothesis, or any final scientific claim. All design
decisions, all numerical results, and all interpretation in this paper
reflect the author's own judgment and responsibility.
